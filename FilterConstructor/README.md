Building

```bash
$ git clone git@github.com:iyegoroff/react-native-image-filter-kit.git
$ cd react-native-image-filter-kit/FilterConstructor
$ npm i
$ mono .paket/paket.exe install
$ npm run restore
$ npm start
```

Running on Android

```bash
$ cd react-native-image-filter-kit/FilterConstructor
$ react-native run-android
```
